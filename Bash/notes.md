# Comandos principales

Bash (Bourne Again SHell) es un interprete de comandos para interactuar con el sistema operativo Unix, como Linux y macOS.

## Carpetas o ficheros

Crear una carpeta

```bash
mkdir mi_carpeta
```

Entrar a una carpetacd mi_carpeta

```bash
cd mi_carpeta
```

Salir de una carpeta

```bash
cd ..
```

## Archivos

Crear un archivo

```bash
touch mi_archivo
```

Borrar un archivo

```bash
rm mi_archivo 
```
